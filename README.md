# 🌸 Amani's Web Development Portfolio

Welcome to my corner of the internet! This repository showcases my journey from a coding enthusiast to a budding Full-Stack Developer.

## ✨ About the Project
This is a personal portfolio website designed to highlight my technical skills and the projects I completed during my Software Development Boot Camp. The site features a sleek, responsive design with a **soft rose and magenta theme**.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom styling featuring smooth transitions and a pink-inspired palette.
* **JavaScript:** Interactive elements including a dynamic slideshow and contact form logic.

## 🚀 Key Projects
* **Tic-Tac-Toe:** A classic logic-based game.
* **Pizza Menu:** A functional UI for data display.
* **Interactive Calculator:** Bridging the gap between math and UI.

## 🎨 Design Philosophy
My portfolio reflects my belief that code should be as beautiful as it is functional. I utilized a "Lavender Blush" and "Deep Rose" color scheme to create a unique, professional, yet personal user experience.
